# تیندال - سایت فروشگاه ساز

یک وبسایت برای ساخت فروشگاه آنلاین که با React نوشته شده.

## درباره پروژه

این پروژه یک لندینگ پیج برای یک سرویس فروشگاه‌ساز آنلاینه. کاربرا میتونن از طریق این سایت با خدمات آشنا بشن و فروشگاه خودشون رو بسازن.

### چیزایی که توی پروژه استفاده شده:

- React برای ساخت UI
- React Router برای صفحات مختلف
- Bootstrap و React Bootstrap برای استایل
- Swiper برای اسلایدرها
- فونت فارسی IRANYekan

## نصب و اجرا

اول پکیج‌ها رو نصب کن:

```bash
npm install
```

بعد پروژه رو اجرا کن:

```bash
npm start
```

سایت روی `http://localhost:3000` باز میشه.

برای ساخت نسخه نهایی:

```bash
npm run build
```
